\section{Introduction}\label{introduction}

This report explores post-COVID economic recovery in selected Asian
countries, using GDP growth, employment rate, and GDP per capita as
indicators. The study uses data from the World Bank World Development
Indicators (WDI), covering the years 2015--2023. We analyze and compare
recovery trajectories using descriptive statistics and data
visualization.

\section{Data Description}\label{data-description}

The data was retrieved using the \texttt{wbdata} Python package from the
World Bank API. We focused on the following indicators:

\begin{itemize}
\tightlist
\item
  \textbf{GDP per capita (constant 2015 US\$)} ---
  \texttt{NY.GDP.PCAP.KD}
\item
  \textbf{GDP growth (annual \%)} --- \texttt{NY.GDP.MKTP.KD.ZG}
\item
  \textbf{Employment to population ratio (\%, age 15+)} ---
  \texttt{SL.EMP.TOTL.SP.ZS}
\end{itemize}

Countries included: China, India, Indonesia, Vietnam, Philippines, and
Japan.
